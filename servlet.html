    Servlet

    Created for Knoledge Media Architectures this project combined HTML, CSS, XML, JavaScript, and MySQL to create a simple servlet. It used XSLT to layout the page and CSS to style it. The JavaScript and XML were connected so that the JavaScript would load the xml and display the Google map and the table with information from the XML. Then the servlet was queried, requesting images from geolocations stored in the XML, and then returned results and generated markers with locations and photos retrieved from Flickr.

    From this I learned about web technologies and the internet communication protocols and how they are structured and used. I also learned the basics of combining multiple web technologies from HTML to CSS to JavaScript.
